HTML Tags:

  1. Heading tag is the heading tag with <h1> beign the bigest heading to <h6> beign the smallest heading.

    2. Eg.

    <h1>This is a heading</h1>

    There six kinds of heading value starting from <h1> to <h6>.

  2. <p> tag: This is for writing paragraphs.

    3. Eg.

    <p>This is a pragraph.</p>
  3. <br> tag is used to produce vertical spacing in paragraphs, block etc.

    4. Multiple <br> tag can be used to produce more vertical spacing.

    Eg.

    <br><br>
  4. <hr> tag is the horizontal ruler for draving horizonal line

    5. Eg.

    <hr><hr>
  5. <a> tag is used to creat links.

    6. Eg.

    <a href="www.link.com">Click Me!</a>

    Using # inside the href will bring to the current page.

    We can even use target attribute to make the page open in a new tab

    Eg.

    <a href="www.link.com", target="blank">Click Me!</a>

    We can even use buttons

    Eg.

    <a href="www.link.com", target="blank"><button>Click Me!</button></a>
  6. <img> tag is used to insert image file into the page.

    7. Eg.

    <img src="images/filename.png", alt="MyLogo", width="104" height="142">

    Here src attribute is the source address, alt is used to provid alternate text in case the image fail to load, width, height are width and height attribute to abjust image size.

  7. <ol>, <ul> are the ordered(number) and unordered(symbol) list respectively.

    8. Eg.

    	<ul>
		<li>First</li>
		<li>Second</li>
		<li>Third</li>
	</ul>
